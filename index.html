Photo Puzzle

Photo Puzzle is a fun filled, additive IOS game which is played by re arranging the 16 broken parts of an image to form a complete picture.The 16 small images of the picture are all generated programatically. The play time for the game is recorded.The app is built with Swift language and uses the IOS's Swipe Gesture Recogonition to move the images in the game.

Community Help

Community Help is a web app built to help people in need. Users of this app can pin their location to indicate a homeless person in their vicinity with a description . People living in the same area , using this app can utilize the information to donate food, clothes etc.. to such people.

The app is built with Python language, MySql Database, EJS for client side templating, Google Maps API and Twitter Bootstrap.

The SpongeBob - Squidward game

The SpongeBob - Squidward IOS game is a simple fun game in which SpongeBob always tries to escape from SquidWard.For every successful escape, SpongeBob gets a point. The moment he fails and touches Squidward the game ends.The user can the restart the game.

The game is built with SpriteKit framework in Swift language. The movement of SpongeBob in the game is implemented through the IOS Core motion Framework.

Locate your Friend

Locate your friend is an IOS app used to trace your friend's location in real time. In situations where there is a large crowd of people like at concerts, busy downtowns or new areas this app will be very helpful. The app also shows your current location, making it easier to locate your friends location with respect to yours.

The real time location tracking is facilitated by sockets. The app is built on Swift language and uses Google Maps API to display the position on a map and also to get geo-coordinates.
